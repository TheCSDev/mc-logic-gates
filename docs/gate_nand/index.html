    NAND gate

    The NAND gate's output is always opposite to the AND gate's output.

    Obtaining

    This block can be broken with any tool, as well as the fist.
    It can be crafted using the following crafting recipe:


    -AND gate-
    -NOT gate-
    ---

    Behavior

    The NAND gate has two inputs and one output. The output is similar to that of AND gate, except it is always opposite to the AND gate's output.

    Data values

    Block ID
