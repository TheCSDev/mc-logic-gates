    NOR gate

    A logic gate whose output is always opposite to that of OR gate's.

    Obtaining

    This block can be broken with any tool, as well as the fist.
    It can be crafted using the following crafting recipe:


    -NOT gate-
    -OR gate-
    ---

    Behavior

    The NOR gate has two inputs and one output. The output is always opposite to that of OR gate's.

    Data values

    Block ID

    logicgates:nor_gate

    Block states

    Blockstate Values Description
    facing east, west, north, south The direction the block is facing.
    powered true, false Whether the block is recieving redstone power.
