    OR gate

    A logic gate whose output is on, so long as one or both of it's inputs are on.

    Obtaining

    This block can be broken with any tool, as well as the fist.
    It can be crafted using the following crafting recipe:


    AND gateNOT gate-
    NOT gate--
    ---

    Behavior

    The OR gate has two inputs and one output. The output is output is on so long as one or both of it's inputs are on.

    Data values

    Block ID

    logicgates:or_gate

    Block states

    Blockstate Values Description
    facing east, west, north, south The direction the block is facing.
    powered true, false Whether the block is recieving redstone power.
    swapped_direction true, false Whether the block's side input direction is swapped.
